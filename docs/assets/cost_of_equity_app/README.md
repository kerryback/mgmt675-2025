# Cost of Equity Analysis App

This Streamlit app calculates the cost of equity for any publicly traded company and generates 
a PowerPoint presentation with the analysis.

## Installation

1. Unzip the files to a directory
2. Install the requirements:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the Streamlit app:
   ```
   streamlit run app.py
   ```
2. Enter a stock ticker (e.g., AAPL, MSFT, WMT)
3. Click "Generate Analysis"
4. View the results and download the PowerPoint presentation

## Components

- `app.py`: Main Streamlit application
- `cost_of_equity_analysis.py`: Core analysis functions
- `requirements.txt`: Required Python packages
