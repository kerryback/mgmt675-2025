\begin{frame}{Setting up a simulation in Julius/python}
\begin{itemize}
\item Model calculates outputs given inputs
\item Ask Julius to create a function that calculates outputs given inputs
\item Ask Julius to run a given number of simulations drawing inputs from given distributions
\item Ask Julius to describe the distributions of the outputs
\begin{itemize}
\item Histogram
\item Mean
\item Median
\item Standard deviation
\item 90\% confidence interval, etc.
\end{itemize}
\end{itemize}
\end{frame}

\begin{frame}{Feeding an Excel model to Julius}
\begin{itemize}
\item We can upload an Excel file to Julius
\item Julius' default mode is to assume each worksheet is a single table.  It will create a "dataframe" for each worksheet containing the numbers in the worksheet.
\item Usually, we have multiple things in a single worksheet - inputs and outputs.
\item Ask Julius to read the formulas and warn Julius that there are multiple tables.
\end{itemize}
\end{frame}
